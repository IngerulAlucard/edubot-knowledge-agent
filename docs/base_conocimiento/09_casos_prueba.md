# Casos de Prueba para EduBot Knowledge Agent

## Caso 1

Pregunta:
¿Cómo puedo obtener mi certificado?

Respuesta esperada:
El estudiante debe completar el 100% del contenido obligatorio, aprobar las evaluaciones, entregar proyecto final si aplica, no tener pagos pendientes y cumplir el reglamento. El certificado se descarga en PDF desde el panel del estudiante.

## Caso 2

Pregunta:
¿Cuánto tiempo tengo para pedir reembolso?

Respuesta esperada:
El estudiante puede solicitar reembolso dentro de los primeros 7 días naturales posteriores a la compra, siempre que no haya completado más del 20% del curso.

## Caso 3

Pregunta:
¿Qué cursos son de Cloud Computing?

Respuesta esperada:
El agente debe consultar el CSV y mencionar cursos como Fundamentos de Cloud Computing, Introducción a Oracle Cloud Infrastructure y Deploy de Aplicaciones en la Nube.

## Caso 4

Pregunta:
¿Cuánto cuesta Python Básico?

Respuesta esperada:
El curso Python Básico cuesta 999 MXN.

## Caso 5

Pregunta:
¿Qué hago si olvidé mi contraseña?

Respuesta esperada:
Debe usar la opción “Recuperar contraseña” desde la pantalla de inicio de sesión. Si el correo no llega, debe revisar spam o contactar a soporte técnico.

## Caso 6

Pregunta:
¿Cuántos intentos tengo para una evaluación?

Respuesta esperada:
Cada evaluación tiene hasta 3 intentos disponibles y el sistema conserva la calificación más alta.

## Caso 7

Pregunta:
¿Puedo usar inteligencia artificial en mis tareas?

Respuesta esperada:
Sí, EduNova Academy permite usar IA como apoyo, pero el estudiante debe comprender, revisar y adaptar su entrega.

## Caso 8

Pregunta:
¿Qué pasa si mi cuenta fue bloqueada?

Respuesta esperada:
Debe contactar a soporte técnico y verificar su identidad. La cuenta puede bloquearse por intentos fallidos, actividad sospechosa o incumplimiento del reglamento.

## Caso 9

Pregunta:
¿Cuánto tarda soporte en responder?

Respuesta esperada:
El tiempo promedio de respuesta es de 24 a 48 horas hábiles.

## Caso 10

Pregunta:
¿Las becas cubren el 100%?

Respuesta esperada:
No. Las becas son parciales y cubren entre el 20% y el 70% del costo del curso.
